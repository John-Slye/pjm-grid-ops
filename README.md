# PJM Grid Ops — Peak-Day Early-Warning System

A peak-day early-warning system for PJM: it predicts which summer days will
rank among the season's five highest demand peaks, backtested over seven
summers, with an expected-savings model for demand curtailment.

> History: the repository began as a demand-forecast evaluation project and
> pivoted to peak-day ranking. See [DECISIONS.md](DECISIONS.md) for the design
> record behind that framing.
